# Tài Liệu Nguồn Tham Chiếu

## 1) Tài liệu trong dự án

- Tài liệu thuyết trình/thực hành:
  - `docs/presentation_guide.md`
- Mã nguồn chính:
  - `app.py`
  - `templates/index.html`
- Bộ kiểm thử:
  - `tests/test_security_lab.py`
- Bài tập fix-me:
  - `exercises/fix_me_sqli.py`
  - `exercises/fix_me_xss.py`

## 2) Tài liệu chuẩn OWASP/CWE

- OWASP Top 10
  - https://owasp.org/www-project-top-ten/
- OWASP SQL Injection
  - https://owasp.org/www-community/attacks/SQL_Injection
- OWASP XSS
  - https://owasp.org/www-community/attacks/xss/
- OWASP XSS Prevention Cheat Sheet
  - https://cheatsheetseries.owasp.org/cheatsheets/Cross_Site_Scripting_Prevention_Cheat_Sheet.html
- OWASP SQL Injection Prevention Cheat Sheet
  - https://cheatsheetseries.owasp.org/cheatsheets/SQL_Injection_Prevention_Cheat_Sheet.html
- CWE-89 (SQL Injection)
  - https://cwe.mitre.org/data/definitions/89.html
- CWE-79 (Cross-Site Scripting)
  - https://cwe.mitre.org/data/definitions/79.html

## 3) Tài liệu kỹ thuật bổ trợ

- SQLite application-defined functions (dùng cho `sleep()` trong lab blind SQLi)
  - https://www.sqlite.org/appfunc.html
- Python `html.escape`
  - https://docs.python.org/3/library/html.html
- Python `json.dumps`
  - https://docs.python.org/3/library/json.html
- Python `urllib.parse.quote`
  - https://docs.python.org/3/library/urllib.parse.html
